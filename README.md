 Sliding Panel Login Page

An animated sliding panel login/signup page built with vanilla HTML, CSS, and JavaScript.

Live Site
[View Live Site](https://luhmmie.github.io/sliding-login-page)

 Features
- Sliding panel animation on toggle
- Sign In and Sign Up forms
- Social login icons (Google, LinkedIn, GitHub)
- Purple gradient overlay panel
- Smooth CSS transitions
- Responsive design (mobile + desktop)
- Slides left/right on desktop, up/down on mobile

 Tech Stack
- HTML5
- CSS3 (Flexbox, transitions, transforms, media queries)
- Vanilla JavaScript (classList, event listeners)

 How it works
The page has two panels — a white form side and a purple gradient overlay side. Clicking Sign Up slides the overlay to the left revealing the registration form. Clicking Sign In slides it back.
